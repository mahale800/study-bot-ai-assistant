# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - Initial Production Release
### Added
- **Cinematic UI/UX:** Interactive glassmorphism panels, neural load/confidence vitals simulation, and dynamic JARVIS-style cursor.
- **3D Background:** Integrated Three.js deep-space parallax background reactive to mouse movement.
- **Voice System:** Native browser `SpeechRecognition` input and `speechSynthesis` output with intelligent text cleaning for natural cadence.
- **AI Backend:** LangGraph and FastAPI orchestration routing requests to Groq for near-instant low latency responses.
- **Threaded Memory:** Persistent local JSON storage (`memory.json`) supporting multiple distinct study sessions.
- **Incognito Mode:** Temporary memory bypass mode with distinct purple visual indicators.
- **Academic Alignment:** Refined system backend instructions tailored specifically to intelligent academic tutoring.
