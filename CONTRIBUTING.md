# Contributing to Study Bot (CYPHER)

Thank you for your interest in contributing to the Study Bot AI Assistant! 

## How to Contribute

1. **Fork the repository** on GitHub.
2. **Clone your fork locally**:
   ```bash
   git clone https://github.com/your-username/study-bot-ai-assistant.git
   ```
3. **Create a feature branch**:
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Make your changes**. Ensure that your code is clean, readable, and does not break existing features (such as the Voice AI, Memory sync, or the 3D WebGL background).
5. **Test locally** by running the FastAPI server and verifying functionality in the browser.
6. **Commit your changes** with clear and descriptive commit messages.
7. **Push to your fork** and open a Pull Request against the `main` branch of the original repository.

## Reporting Bugs

Please use the GitHub Issues tab to report any bugs. Provide:
- A clear description of the issue
- Steps to reproduce the behavior
- Console or terminal Error logs if applicable

## Feature Requests

Have a great idea? Open an Issue tagged as an `enhancement` and describe what the feature is and how it would improve the system.
